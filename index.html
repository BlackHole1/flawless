居中样式表

我是内容

在div里加上class="container"就行，例： <div class="container">我是内容</div> ，多层container不会起作用，因为container使用的是px绝对定位，而不是%相对定位

占位符

我的上下左右全部被占了5px空间
placeholder-top是向上占5px，同理placeholder-bottom placeholder-left placeholder-right各向下、左、右占位，例： <div class="placeholder-top placeholder-bottom placeholder-left placeholder-right">我的上下左右全部被占了5px空间</div>

A标签 外链

自定义内容
在a标签里加入class="a_tag"即可。如果你没有在a标签里添加任何内容，就像第一个例子一样自动添加内容。
代码：<a href="#" class="a_tag"></a> 超链接图标、管道符、title都会由jquery自动添加

在a标签里加入class="a_tag"时，如果你想自定义a标签内容，可直接填写内容。就像第二例子一样
代码：<a href="#" class="a_tag">自定义内容</a> 程序不会自动帮你添加a标签内容了

A标签 幽灵按钮

幽灵按钮
在a标签里加入class="a_tag-button"即可。 <a href="#" class="a_tag-button">幽灵按钮</a>

幽灵按钮2


在button标签里加入class="button_tag"即可。例： <button class="button_tag">START</button> ，就像第一个例子。
加入data即可控制大小，例： <button class="button_tag" data="70">START</button> ，就像第二个例子。

img图片样式

当class="img-circle"时，是普通圆形样式。就像第一个。
当class="img-circle-hover"时，圆形带边框和变化效果，例子为第二个。
当class="img-frames"时，带边框圆角样式，例子为第三个
当class="img-frames-hover"时，带边框圆角样式和变化效果，例子为第四个

form标签及input 例子1

在form标签里加入class="form_tag"即可。例： <form action="#" class="form_tag"> ，input样式只需要在input里插入data就行了。例： <input type="text" data="email" /> ，data为email、邮箱时,就像第一个例子。当data为user、username、用户、用户名、登录名时，就是第二个例子。当data为password、密码时，就是第三个例子。
如果没有写data，例： <input type="text" /> ,将会自动帮你添加字符串为“搜索/Search”。如果你不写type，例： <input />,也会自动帮你添加字符串为“搜索/Search”，就像第四个
如果你想自定义显示input的内容，在input标签里加上data属性，值添上你所想显示的内容就行(不能和"email", "邮箱", "user", "username", "用户", "用户名", "登录名", "password", "密码"一样)，例： <input type="text" data="我是自定义内容" /> 。就像第五个例子一样
在input添加width、height即可自定义input的宽高(防止有些人不会css...)，例： <input type="text" data="自定义宽高" width="200" height="30" />

代码片段

I'm code fragment;
在code标签里加入class="code-fragment"即可。例： <code class="code-fragment">code</code>

代码

$("code").each(function() {
	if (!$(this).hasClass('code-fragment')) {
		$(this).addClass('code_tag');
	}
});
直接写code标签就行，无需加其他样式，如果code样式中存在code-fragment，将不会起作用。例： <code>i'm code</code>
