消息窗口

I'm 警告信息
I'm 注意事项

一个div包含span。图标自动添加，div的样式为alert-news，警告、注意、信息、成功的样式分别为error、warning、info、success，例： 
<div class="alert-news error">
  <span>I'm 警告信息</span>
</div>
 在增加一个class样式为close，就可以实现点击X渐渐关闭，如 <div class="alert-news error close">*******</div>

A标签 外链

自定义内容
在a标签里加入class="a-tag"即可。如果你没有在a标签里添加任何内容，就像第一个例子一样自动添加内容。
代码：<a href="#" class="a-tag"></a> 超链接图标、管道符、title都会由jquery自动添加

在a标签里加入class="a-tag"时，如果你想自定义a标签内容，可直接填写内容。就像第二例子一样
代码：<a href="#" class="a-tag">自定义内容</a> 程序不会自动帮你添加a标签内容了

A标签 幽灵按钮

幽灵按钮
在a标签里加入class="a-tag-button"即可。 <a href="#" class="a-tag-button">幽灵按钮</a>

幽灵按钮2


在button标签里加入class="button-tag"即可。例： <button class="button-tag">START</button> ，就像第一个例子。
加入data即可控制大小，例： <button class="button-tag" data="70">START/button> ，就像第二个例子。

按钮组


页面按钮组(A/button)

1 2 3 4 5



在div里加入class="pagegroup",再在div里放入a标签就行了。同理在div里加入class="a-page，再里面放入button标签就行了。
如果想在当前页面的按钮不可点，在A标签或则button里加上class="new"就行了。凡是加上new样式的，显示的都是不可点状态
当.new样式(当前页面)是第一页或则最后一页的时候，<<和>>自动不可点
A标签按钮组例子： 
<div class="a-page">
  <a href="#" class="new">1</a>
  <a href="#">2</a>
  <a href="#">3</a>
  <a href="#">4</a>
  <a href="#">5</a>
</div>
 button标签按钮组例子： 
<div class="button-page">
  <button class="new">1</button>
  <button>2</button>
  <button>3</button>
  <button>4</button>
  <button>5</button>
</div>

当class="img-circle-hover"时，圆形带边框和变化效果，例子为第二个。
当class="img-frames"时，带边框圆角样式，例子为第三个
当class="img-frames-hover"时，带边框圆角样式和变化效果，例子为第四个

input样式









在form标签里加入class="form-tag"即可。例： <form action="#" class="form-tag"> ，input样式只需要在input里插入data就行了。例： <input type="text" data="email" /> ，data为email、邮箱时,就像第一个例子。当data为user、username、用户、用户名、登录名时，就是第二个例子。当data为password、密码时，就是第三个例子。
如果没有写data，例： <input type="text" /> ,将会自动帮你添加字符串为“搜索/Search”。如果你不写type，例： <input />,也会自动帮你添加字符串为“搜索/Search”，就像第四个
如果你想自定义显示input的内容，在input标签里加上data属性，值添上你所想显示的内容就行(不能和"email", "邮箱", "user", "username", "用户", "用户名", "登录名", "password", "密码"一样)，例： <input type="text" data="我是自定义内容" /> 。就像第五个例子一样
在input添加width、height即可自定义input的宽高(防止有些人不会css...)，例： <input type="text" data="自定义宽高" width="200" height="30" />

想让几个input紧紧相邻的话，可以给他们一个class，左边的给 class="group-left" ，右边的给 class="group-right" ，中间的给 class="group-all"
代码实例： 
<form action="#" class="form-tag">
　　<input class="group-left"type="text" data="user">
　　<input class="group-all" type="password">
　　<input class="group-all" type="password" data="确认密码">
　　<input class="group-right" type="submit">
</form>

边框颜色

在div标签里加入class="left-border-color"即可(left-border-color和embed更搭哦...)。例： <div class="embed left-border-color">欧巴，你的left-border-color。“是你的left-border-color”</div>

代码片段

I'm code fragment;
在code标签里加入class="code-fragment"即可。例： <code class="code-fragment">code</code>

代码

$("code").each(function() {
  if (!$(this).hasClass('code-fragment')) {
    $(this).addClass('code_tag');
  }
});
直接写code标签就行，无需加其他样式，如果code样式中存在code-fragment，将不会起作用。例： <code>i'm code</code>
